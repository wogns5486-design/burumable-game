======================================
위시켓 지원서 - 팝업스토어 부루마블 웹게임
======================================

【지원 금액】
2,500,000

【지원 기간】
10

【클라이언트 질문 답변】
"부루마블같은 말판 이용시키는 주사위게임을 만들어보신적이 있으신가요?"

네, 부루마블 형태의 보드게임 웹 개발 경험이 있습니다.

실제로 팝업스토어 이벤트용 부루마블 웹게임을 개발한 바 있으며, 현재 동작하는 데모를 보유하고 있습니다.

구현 완료된 주요 기능:
- 9x9 그리드 기반 32칸 부루마블 보드판 (세계 도시 테마)
- CSS 3D 주사위 2개 동시 굴림 + 회전 애니메이션
- 말의 칸별 바운스 이동 애니메이션
- 황금열쇠/찬스 등 랜덤 이벤트 시스템
- 더블 판정, 무인도 1턴 쉼, 출발 통과 보너스 등 핵심 룰
- PC/태블릿/모바일 반응형 대응
- 파티클 이펙트, 카드 플립 애니메이션 등 풍부한 인터랙션

미팅 시 실제 동작하는 데모를 바로 보여드릴 수 있습니다.


【지원 내용】

<프로젝트 진행 제안>

이미 부루마블 웹게임의 핵심 기능을 개발 완료한 상태이며, 클라이언트님의 팝업스토어 브랜드와 컨셉에 맞게 커스터마이징하여 빠르게 납품할 수 있습니다.

<유사 프로젝트 진행 경험>

부루마블 형태의 보드게임 웹 개발 경험 보유. 로그인 없이 즉시 플레이 가능한 이벤트용 웹게임 구현 완료. CSS 3D 애니메이션 기반 주사위, 파티클 이펙트 등 고퀄리티 인터랙션 구현. PC/모바일/태블릿 반응형 웹 대응 경험 다수.

<사용 기술과 툴>

프론트엔드: HTML5, CSS3 (3D Transform, Keyframes), Vanilla JavaScript
디자인: Figma
반응형: CSS Grid + clamp() 기반 fluid 레이아웃
배포: Cloudflare Pages
별도 라이브러리 의존성 없이 순수 웹 기술로 구현하여 유지보수가 간편하고 로딩 속도가 빠릅니다.

진행 일정 (10일):
- 1~2일차: 클라이언트 미팅 + 브랜드 컨셉 반영 기획
- 3~5일차: UI/UX 디자인 + 보드 커스터마이징
- 6~8일차: 프론트엔드 개발 + 애니메이션 구현
- 9일차: QA 및 버그 수정
- 10일차: 배포 및 납품

미팅 시 실제 동작하는 데모를 바로 시연해드릴 수 있어, 완성 퀄리티를 사전에 확인하실 수 있습니다.
